# tmp-detailed-claim-event contracts

All the files and directories within this directory have been generated using the import-contracts command of the CLI in this repo.
The ABI and the binnaries of the smart contracts have been extracted from [agglayer-contracts repo](https://github.com/agglayer/agglayer-contracts.git), using the version ecdde63f79f2d0264d328b5983aec9b6f532b656 (commit ecdde63f79f2d0264d328b5983aec9b6f532b656)

The CLI command used to generate the contracts: `$ go run ./cmd import-contracts -cv ecdde63f79f2d0264d328b5983aec9b6f532b656 -alias tmp-detailed-claim-event -node 22`
